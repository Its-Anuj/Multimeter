# Multimeter
A program which displays voltage and current from a circuit in a 16x2 lcd display using arduino

## Equiment Required

- 1 16x2 LCD display
- 1 10k potentiometer
- Jumper Wires (or any wires for connecting the circuit)
- Arduino Uno R3
